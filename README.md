# clone-tabnews

Implementação do https://www.tabnews.com.br para o https://curso.dev

## Visão Geral das Tecnologias Utilizadas

| Tecnologia                           | Função                                             |     |
| ------------------------------------ | -------------------------------------------------- | --- |
| **JavaScript (ES6+)**                | Linguagem principal                                |     |
| **React.js**                         | Construção da interface                            |     |
| **Next.js**                          | Estrutura do projeto + SSR (Server-Side Rendering) |     |
| **npm**                              | Gerenciador de pacotes                             |     |
| **Vercel**                           | Deploy da aplicação                                |     |
| **Prettier / ESLint**                | Padronização e qualidade de código                 |     |
| **Git & GitHub**                     | Controle e versionamento de código                 |     |
| (Opcional) **Jest/Tests**            | Testes automatizados                               |     |
| (Opcional) **CI/CD**                 | Integração contínua                                |     |
| (Opcional) **Docker**                | Ambiente container                                 |
| Meu site ([victoralmeidadev.com][1]) |
